Moduino 라이브러리 함수 설명

init() : 아두이노 환경에 필요한 설정 및 초기화가 이루어지는 함수입니다.
FindID() : 연결된 모듈 전체에 모듈 정보 송신 요청을 보냅니다.
RequestProperyt(Module*) : 연결된 모듈의 신호 출력 활성화 요청을 보냅니다.
ReceiveProcess() : 모듈 신호 수신 처리 함수입니다.


사용순서

1. 연결한 모듈의 정보를 파악하기 위해 setup()에서 FindID()를 호출 후 Serial Monitor를 이용하여 모듈들의 ID를 확인합니다.

2. 전역 변수로 모듈별 객체 포인터를 선언 후 addModule(ID)를 호출하여 객체 포인터에 저장합니다.

3. 입력모듈이 있으면 RequestProperty(Module*)를 호출하여 입력 수신을 활성화합니다.

4. void loop()에 코드를 작성 및 아두이노에 업로드하여 실행합니다.


사용 가능 모듈 목록 및 모듈별 호출가능 함수

LED : setRGB(red,green,blue)

다이얼 : getDegree()

버튼 : getClick()


주의할 점

1. void loop()에서 반드시 ReceiveProcess()를 포함하여야 하며 지우면 코드가 실행되지 않습니다.

2. void loop()의 첫 문장은 가급적 ReceiveProcess()가 되도록 하며 Delay는 사용하면 안됩니다.

3. setup()의 첫 문장은 반드시 modi.init()이 되도록 합니다.

3. FindID() 사용 시 가급적 setup()의 마지막에 적어주도록 합니다.
